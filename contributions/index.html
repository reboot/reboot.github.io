Snes9x Joystick Axis Selection

With this patch you can specify which joystick axis should be mapped to the directions of the SNES gamepad in the Snes9x SNES emulator. The original code has axis 0 and 1 hardcoded and that was a analog stick on my gamepad which was not very good to use for the digital SNES gamepad. With this patch I could select the digital pad on my gamepad for this.

Project Homepage http://www.snes9x.com/
Patch snes9x.patch
Submission date 5 Sep 2005
Status Waiting for answer from author


Liferea IPv6 support

This patch allows Liferea to connect to IPv6 hosts or proxy servers, by using the better getaddrinfo() function that returns sockinfo-structures for any kind of network protocol.

Project Homepage http://liferea.sourceforge.net/

Submission date 4 Sep 2004
Status Included in Liferea 0.6.1


MaCoPiX Socket Message Feature

This patch allows to send messages to the MaCoPiX window sitter that are then displayed in the mascot's balloon message. This can be used for various purposes. Every program that can run an external command for events can use this to make the mascot say a text. I use this for new mail notification from Sylpheed-Claws.

Project Homepage http://kurumi.sakura.ne.jp/~chimari/macopix/index-e.html
Patch macopix-sockmsg.patch
Submission date 6 Jul 2004
Status Included in MaCoPiX 1.2.0


GKrellDnet Multi CPU Krells
